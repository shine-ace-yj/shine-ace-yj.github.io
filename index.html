作用域-1：是什么？

发表于 2017-04-27 | 分类于 你不知道的Javascript 上

作用域

引擎：负责Javascript程序的执行和编译过程；
编译器：负责语法分析及代码生成等；
作用域：负责收集并维护所有声明的标识符即变量的查询和确定执行代码对这些标识符的访问权限；

  1. 变量的赋值操作var a = 2; 执行2个动作，一，编译器询问作用域是否存在了a这个名称的变量，存在，就忽略var a这个操作，否则就在当前作用域中声明该变量并命名为a，二，在运行时引擎会在作用域中查询该变量，如果查找到了该变量，就对它进行赋值操作；

  2. “ LHS查询 ” 和 “ RHS查询 ”：RHS查询某个变量的值，LHS找到变量的容器本身进行赋值

    function foo (a) {
  console.log(a);
}
foo(2);

    (1) 引擎在作用域中查询’foo’进行RHS引用,作用域返回结果“foo”是一个函数；
    (2) 引擎在作用域中查询’a’进行LHS引用,作用域返回结果“a”是一个形式参数；
    (3) 引擎将2赋值给参数a，console RHS引用，a->RHS引用;

  3. 作用域嵌套：引擎从当前的执行作用域进行查找，如果找不到就向上一级作用域继续查找，直到到达全局作用域停止；

  4. 异常：ReferenceError: 在作用域中查询不到该变量是抛出； TypeError: 作用域判别成功了，即找到了该变量，但对结果的操作是非法的；

  5. 词法作用域：定义在词法阶段的作用域，由代码中变量和作用域的位置决定，无论函数在哪里被调用，函数的作用域都只由函数定义时的位置决定；

函数作用域

  1. 函数作用域指的是函数中的全部变量可以在函数中的任意地方使用；
  2. 区分函数表达式和函数声明的方法是 function 关键字出现的位置是否是第一位，如果是，则为函数声明，否则为函数表达式；

  3. IIFE -> 立即执行函数表达式 

    ( function ( ) { } )( )
+/- function( ) { } ( )

    第一个括号表示函数为表达式，第二个表示为函数调用，可以传入参数;

###块级表达式

  1. try/catch 中的catch为块级表达式，其声明的变量只在catch中有用；
  2. let/const

###声明提升

  1. var a = 2被分为2个片段执行var a和a = 2，第一个为编译阶段执行，后一个为运行阶段执行；
  2. 无论声明在什么地方，都将在代码本身被执行前进行处理，所有声明都会被移动至作用域最前端，而赋值操作会留在原地按序执行；
  3. 函数提升优先于变量提升，变量与函数同名时；
